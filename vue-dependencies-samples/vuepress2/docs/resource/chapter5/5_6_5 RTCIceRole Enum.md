### 5.6.5 RTCIceRole枚举

```java
enum RTCIceRole {
    "controlling",
    "controlled"
};
```

| RTCIceRole枚举描述 |                                                  |
| ------------------ | ------------------------------------------------ |
| `controlling`      | 一个由[ICE]，第三节中定义的`controlling agent`。 |
| `controlled`       | 一个由[ICE]，第三节中定义的`controlled agent`。  |
